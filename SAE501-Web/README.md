# SAE5.01 #
Flexitrip est une application conçue pour prendre en charge les personnes à mobilité réduite (PMR) lors de leurs déplacements, qu'ils soient de courte ou de longue distance.
Ce qui offre une solution adaptée pour rendre leurs voyages plus simples et accessibles.

Fonctionnalités :
- Notifications en temps réel : Services gérés via une API communiquant à travers un hub de messagerie (Kafka).
- Réservations multimodales : Création d’un e-billet unique pour tous types de transports (train, avion, voiture, etc.).
- Paiement interne sécurisé : Un système de crédit décentralisé alimenté par la blockchain.
- Enregistrement des bagages : Un système simple d’utilisation pour vérifier et confirmer les bagages des personnes à mobilité réduite (PMR).
